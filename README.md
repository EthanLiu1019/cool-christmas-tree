# 🎄 Cool Christmas Tree

一个用 **HTML + CSS + JS** 写的炫酷圣诞树网页，每年圣诞节都会更新一次！

## 功能
- 三层绿色树冠，顶部金色星星闪烁
- 彩色装饰球自动分布并闪烁
- 雪花不断飘落，大小和速度随机
- 可直接部署到 GitHub Pages

## 使用方法
1. 打开 'index.html' 即可在浏览器中看到效果。
2. 或者上传到 GitHub 仓库并启用 **GitHub Pages**，通过链接访问。

## 部署到 GitHub Pages
1. 在 GitHub 创建仓库 'cool-christmas-tree'
2. 上传所有文件（'index.html', 'style.css', 'script.js', 'README.md'）
3. 打开仓库 → Settings → Pages → 选择 'main' 分支 → 保存
4. 几分钟后访问：https://你的用户名.github.io/cool-christmas-tree
